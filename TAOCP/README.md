The Art of Computer Programming
===============================

高德纳（Donald Ervin Knuth，唐纳德.E.克努特）是算法和程序设计技术的先驱，1974年图灵奖得主，计算机排版系统TeX和字型设计系统Metafont的发明者。

1962年，高德纳开始写作TAOCP。

1968年—1973年，前三卷陆续出版。

1974年，高德纳荣获图灵奖，成为该奖项最年轻的得主。

1976年，写作中的高德纳发现图书排版缺乏美感，于是着手发明TeX和Metafont。

TAOCP的中文版由苏运霖教授翻译。
