# IAT
原子时（International Atomic Time，IAT）以物质的原子内部发射的电磁振荡频率为基准的时间计量系统。

# 概念
原子时的秒长是这样定义的“铯133原子在基态的两个超精细能级结构间零磁场跃迁时，辐射频率的9,192,631,770个周期持续的时间为1秒”。

1958年1月1日世界时的0时为原子时的时刻起点。
